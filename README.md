# PDF TO JPG 변환기

PDF TO JPG 변환기(이하 "P2J")는 PDF를 JPG로 변환하는 프로그램으로서, 추후 PDF 및 JPG 관련 기능들이 추가될 예정입니다.

Function
===
P2J 다음과 같은 기능을 갖고있습니다.
+ PDF를 JPG로 변환해줍니다.
+ PDF를 병합합니다(추가예정).
+ PDF를 분할합니다(추가예정).
+ JPG를 PDF로 변환합니다(추가예정).
